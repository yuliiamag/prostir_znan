# Простір знань

Вебзастосунок для організації індивідуального навчального процесу між вчителем, учнем та батьками.
Основною метою проєкту є спрощення організації індивідуального навчання шляхом об'єднання
всіх необхідних функцій в єдиній платформі та зменшення потреби у використанні кількох окремих сервісів.

## Основні можливості

* Реєстрація та авторизація користувачів
* Ролі вчителя, учня та батьків
* Планування та перегляд занять
* Домашні завдання та перевірка виконання
* Внутрішній чат між учасниками навчального процесу
* Система сповіщень
* Статистика та досягнення учнів
* Синхронізація з Google Calendar
* Проведення онлайн-занять через Google Meet

## Використані технології

* Python
* Django
* PostgreSQL
* Docker
* Docker Compose
* HTML
* CSS
* JavaScript

## Запуск проєкту

Клонування репозиторію:

```bash
git clone https://github.com/yuliiamag/prostir_znan.git
cd prostir_znan
```

Запуск через Docker:

```bash
docker compose up --build
```

Після запуску застосунок буде доступний за адресою:

```text
http://localhost:8000
```
